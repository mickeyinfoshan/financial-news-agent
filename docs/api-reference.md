# API Reference

## Main Function

### `run_agent(user_query: str) -> dict`

**Location**: `financial_news_agent/agent.py:15`

Executes the financial news agent to search for news and generate analysis.

**Parameters:**
- `user_query` (str): The user's question about a company or industry
  - Examples: "How is Tesla performing recently?", "What's happening with NVIDIA stock?"

**Returns:**
- `dict`: Structured result containing answer, sources, evaluation, and trace data

**Example:**
```python
from financial_news_agent.agent import run_agent

result = run_agent("How is Tesla performing recently?")
print(result["answer"])
print(f"Score: {result['evaluation']['overall']}/10")
```

## Response Structure

### Complete Response Object

```python
{
    "answer": str,              # Final analysis with storyline and future impact
    "sources": List[dict],      # News articles used as sources
    "tool_calls": List[dict],   # Tools invoked during execution
    "reasoning_steps": List[str], # LLM reasoning outputs
    "evaluation": dict,         # Self-evaluation scores
    "trace": dict              # Complete traceability data
}
```

### Field Details

#### `answer` (string)
The final analysis generated by the agent, including:
- Summary of recent developments
- Storyline connecting events
- Future impact analysis

**Example:**
```
"Based on recent news, Tesla has been experiencing significant developments 
in the EV market. The company announced record deliveries in Q1 2026, with 
500,000 vehicles delivered globally. However, increased competition from 
Chinese manufacturers has put pressure on margins..."
```

#### `sources` (list of dicts)
News articles retrieved and used in the analysis.

**Structure:**
```python
{
    "title": str,      # Article headline
    "date": str,       # Publication date (ISO format)
    "source": str,     # News source name
    "url": str,        # Article URL
    "summary": str     # Article description/summary
}
```

**Example:**
```python
{
    "title": "Tesla Reports Record Q1 Deliveries",
    "date": "2026-05-15T10:30:00Z",
    "source": "Reuters",
    "url": "https://reuters.com/...",
    "summary": "Tesla Inc reported record vehicle deliveries..."
}
```

#### `tool_calls` (list of dicts)
Record of all tool invocations during execution.

**Structure:**
```python
{
    "tool_name": str,           # Name of the tool
    "arguments": dict,          # Arguments passed to tool
    "result": List[dict]        # Full result from tool
}
```

**Example:**
```python
{
    "tool_name": "search_financial_news",
    "arguments": {
        "query": "Tesla stock performance",
        "days_back": 7
    },
    "result": [...]  # List of articles
}
```

#### `reasoning_steps` (list of strings)
Text outputs from the LLM during the agent loop, showing the reasoning process.

**Example:**
```python
[
    "I need to search for recent news about Tesla to understand their performance.",
    "Based on the articles, I can see three main themes: delivery records, competition, and pricing strategy."
]
```

#### `evaluation` (dict)
Self-evaluation scores and feedback.

**Structure:**
```python
{
    "accuracy": int,        # 1-10: Information matches sources
    "relevance": int,       # 1-10: Sources relate to query
    "coherence": int,       # 1-10: Storyline flows logically
    "reasonableness": int,  # 1-10: Future impact is plausible
    "overall": float,       # Average of four scores
    "feedback": str         # Explanation of scores
}
```

**Example:**
```python
{
    "accuracy": 8,
    "relevance": 8,
    "coherence": 9,
    "reasonableness": 9,
    "overall": 8.5,
    "feedback": "Strong analysis backed by relevant sources. The storyline effectively connects recent events. Future impact assessment is well-reasoned though slightly optimistic."
}
```

#### `trace` (dict)
Complete traceability data combining all tracking information.

**Structure:**
```python
{
    "sources": List[dict],        # Same as top-level sources
    "tool_calls": List[dict],     # Same as top-level tool_calls
    "reasoning_steps": List[str]  # Same as top-level reasoning_steps
}
```

## Tool Schema

### `search_financial_news`

**Location**: `financial_news_agent/news_tool.py:7`

Searches recent financial news via NewsAPI.

**OpenAI Function Schema:**
```python
{
    "type": "function",
    "function": {
        "name": "search_financial_news",
        "description": "Search recent financial news about stocks, companies, or industries",
        "parameters": {
            "type": "object",
            "properties": {
                "query": {
                    "type": "string",
                    "description": "Company name, stock ticker, or industry to search for"
                },
                "days_back": {
                    "type": "integer",
                    "description": "Number of days to search back (default: 7)",
                    "default": 7
                }
            },
            "required": ["query"]
        }
    }
}
```

**Parameters:**
- `query` (string, required): Search term (e.g., "Tesla", "NVDA", "semiconductor industry")
- `days_back` (integer, optional): Days to search back from today (default: 7, max: 30)

**Returns:**
- `List[dict]`: Up to 10 news articles, each containing:
  - `title`: Article headline
  - `description`: Article summary
  - `url`: Article URL
  - `publishedAt`: Publication timestamp
  - `source`: News source object with `name` field
  - `author`: Article author (may be null)
  - `content`: Article content snippet

**Example Usage (by LLM):**
```json
{
    "tool_name": "search_financial_news",
    "arguments": {
        "query": "Tesla stock",
        "days_back": 7
    }
}
```

## Evaluation Criteria

### Scoring Scale: 1-10

All evaluation criteria use a 1-10 scale where:
- **10**: Excellent
- **8-9**: Good
- **6-7**: Acceptable
- **4-5**: Poor
- **1-3**: Very poor

### Criteria Definitions

#### Accuracy (1-10)
**Question**: Does the information in the answer match the sources provided?

**Scoring Guidelines:**
- 10: All claims directly supported by sources
- 8-9: Most claims supported, minor unsupported details
- 6-7: Some claims lack source support
- 4-5: Many claims not backed by sources
- 1-3: Information contradicts sources

#### Relevance (1-10)
**Question**: Are the sources relevant to answering the query?

**Scoring Guidelines:**
- 10: All sources directly address the query
- 8-9: Most sources highly relevant
- 6-7: Some sources tangentially related
- 4-5: Many sources off-topic
- 1-3: Sources unrelated to query

#### Coherence (1-10)
**Question**: Does the storyline flow logically and make sense?

**Scoring Guidelines:**
- 10: Perfect narrative flow, clear connections
- 8-9: Good flow with minor gaps
- 6-7: Understandable but disjointed
- 4-5: Confusing structure
- 1-3: Incoherent or contradictory

#### Reasonableness (1-10)
**Question**: Is the future impact analysis plausible and well-reasoned?

**Scoring Guidelines:**
- 10: Highly plausible, well-supported predictions
- 8-9: Reasonable predictions with good logic
- 6-7: Somewhat speculative but defensible
- 4-5: Questionable assumptions
- 1-3: Implausible or poorly reasoned

#### Overall Score
**Calculation**: Average of the four criteria scores

**Interpretation:**
- **9.0-10.0**: Excellent response
- **7.0-8.9**: Good response
- **5.0-6.9**: Acceptable response
- **3.0-4.9**: Poor response
- **1.0-2.9**: Very poor response

## Traceability System

### TraceabilityTracker Class

**Location**: `financial_news_agent/traceability.py:4`

Tracks all sources, tool calls, and reasoning steps during agent execution.

**Methods:**

#### `add_source(source_data: dict) -> None`
Records a news article as a source.

**Parameters:**
```python
{
    "title": str,
    "date": str,
    "source": str,
    "url": str,
    "summary": str
}
```

#### `add_tool_call(tool_name: str, arguments: dict, result: Any) -> None`
Records a tool invocation.

**Parameters:**
- `tool_name`: Name of the tool
- `arguments`: Arguments passed to tool
- `result`: Return value from tool

#### `add_reasoning(step: str) -> None`
Records an LLM reasoning output.

**Parameters:**
- `step`: Text content from LLM response

#### `get_trace() -> dict`
Returns complete traceability data.

**Returns:**
```python
{
    "sources": List[dict],
    "tool_calls": List[dict],
    "reasoning_steps": List[str]
}
```

## Environment Variables

### Required Configuration

All environment variables must be set in `.env` file:

```bash
# OpenAI-compatible API configuration
OPENAI_API_KEY=<your-api-key>
OPENAI_BASE_URL=<api-endpoint-url>
OPENAI_MODEL=<model-name>

# NewsAPI configuration
NEWS_API_KEY=<your-newsapi-key>
```

### Variable Details

- **OPENAI_API_KEY**: API key for the LLM endpoint
- **OPENAI_BASE_URL**: Base URL for OpenAI-compatible API
- **OPENAI_MODEL**: Model identifier (e.g., "gpt-5.5", "gpt-4-turbo")
- **NEWS_API_KEY**: NewsAPI key (get free at https://newsapi.org)

## Error Handling

### Agent Errors

If the agent encounters an error during execution:

```python
{
    "answer": "Error occurred: <error message>",
    "sources": [],
    "tool_calls": [],
    "reasoning_steps": [],
    "evaluation": {
        "accuracy": 0,
        "relevance": 0,
        "coherence": 0,
        "reasonableness": 0,
        "overall": 0.0,
        "feedback": "Agent execution failed"
    },
    "trace": {...}
}
```

### Evaluation Errors

If evaluation fails, default scores are returned:

```python
{
    "accuracy": 3,
    "relevance": 3,
    "coherence": 3,
    "reasonableness": 3,
    "overall": 3.0,
    "feedback": "Evaluation failed: <error message>"
}
```

## Usage Examples

### Basic Usage

```python
from financial_news_agent.agent import run_agent

# Run agent
result = run_agent("What's happening with NVIDIA stock?")

# Access answer
print(result["answer"])

# Check quality
if result["evaluation"]["overall"] >= 7.0:
    print("High quality response")

# Review sources
for source in result["sources"]:
    print(f"- {source['title']} ({source['source']})")
```

### Accessing Traceability Data

```python
result = run_agent("How is Tesla performing?")

# Tool calls made
print(f"Tools called: {len(result['tool_calls'])}")
for call in result["tool_calls"]:
    print(f"  - {call['tool_name']}: {call['arguments']}")

# Sources used
print(f"Sources found: {len(result['sources'])}")

# Reasoning steps
print(f"Reasoning steps: {len(result['reasoning_steps'])}")
```

### Filtering by Quality

```python
result = run_agent("Semiconductor industry trends")

eval_data = result["evaluation"]

# Check individual criteria
if eval_data["accuracy"] < 6:
    print("Warning: Low accuracy score")

if eval_data["relevance"] < 6:
    print("Warning: Sources may not be relevant")

# Overall quality gate
if eval_data["overall"] < 5.0:
    print("Response quality below threshold")
    print(f"Feedback: {eval_data['feedback']}")
```

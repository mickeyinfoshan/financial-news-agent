# 前端功能验证清单

## 手动测试步骤

### 1. 基础访问测试
- [ ] 打开 http://localhost:5173
- [ ] 页面正常加载，无控制台错误
- [ ] 看到 "Financial News Agent" 标题
- [ ] 看到左侧会话列表（空状态）
- [ ] 看到中间聊天区域（欢迎界面）
- [ ] 看到右侧来源面板（空状态）

### 2. 会话创建测试
- [ ] 点击 "New Session" 按钮
- [ ] 新会话出现在左侧列表
- [ ] 会话被自动选中（高亮显示）
- [ ] 中间区域显示输入框
- [ ] 输入框可以聚焦

### 3. 查询提交测试（需要后端运行）
**前提：** 后端 API 运行在 http://localhost:8000

- [ ] 在输入框输入查询：`Tesla earnings Q1 2026`
- [ ] 按 Enter 键提交
- [ ] 用户消息立即显示（蓝色气泡，右对齐）
- [ ] 看到流式响应开始：
  - [ ] "Thinking..." 指示器
  - [ ] "Searching NewsAPI..." 或 "Searching Finnhub..." 状态
  - [ ] 文字逐字显示（流式效果）
  - [ ] "Evaluating..." 状态
- [ ] 最终响应完整显示

### 4. 引用系统测试
- [ ] 响应中包含引用标记 `[1]`, `[2]`, `[3]` 等
- [ ] 引用显示为蓝色可点击按钮
- [ ] 鼠标悬停在引用上显示工具提示
- [ ] 点击引用 `[1]`：
  - [ ] 右侧来源面板自动打开（如果关闭）
  - [ ] 对应的来源卡片高亮显示（黄色背景）
  - [ ] 自动滚动到该来源
- [ ] 点击来源卡片：
  - [ ] 来源卡片高亮
  - [ ] 响应中对应的引用高亮

### 5. 来源面板测试
- [ ] 右侧显示所有引用的新闻来源
- [ ] 每个来源卡片包含：
  - [ ] 引用编号徽章（如 `[1]`）
  - [ ] 文章标题（可点击）
  - [ ] 来源名称（如 Reuters, Bloomberg）
  - [ ] 发布日期
  - [ ] 文章摘要
  - [ ] API 来源标签（NewsAPI 或 Finnhub）
- [ ] 点击文章标题在新标签页打开原文
- [ ] 来源数量显示正确（如 "Sources (5)"）

### 6. 质量评分测试
- [ ] 响应下方显示质量评分
- [ ] 显示总分（如 "Quality Score: 8.5/10"）
- [ ] 分数颜色正确：
  - [ ] 8-10: 绿色
  - [ ] 6-7.9: 黄色
  - [ ] 4-5.9: 橙色
  - [ ] 0-3.9: 红色
- [ ] 点击评分可展开详细信息：
  - [ ] Accuracy 分数
  - [ ] Relevance 分数
  - [ ] Coherence 分数
  - [ ] Reasonableness 分数
  - [ ] 反馈文本

### 7. 多轮对话测试
- [ ] 提交第二个查询：`What about competition?`
- [ ] 新的用户消息显示
- [ ] 新的 AI 响应流式显示
- [ ] 两条消息都保留在聊天历史中
- [ ] 每条消息有独立的来源和评分
- [ ] 自动滚动到最新消息

### 8. 会话管理测试
- [ ] 创建第二个会话
- [ ] 在第二个会话中提交查询
- [ ] 切换回第一个会话：
  - [ ] 第一个会话的消息正确显示
  - [ ] 来源面板显示第一个会话的来源
- [ ] 删除会话：
  - [ ] 鼠标悬停在会话上显示删除按钮
  - [ ] 点击删除按钮显示确认对话框
  - [ ] 确认后会话从列表中移除

### 9. UI 交互测试
- [ ] 点击左上角菜单按钮切换侧边栏显示/隐藏
- [ ] 点击右上角 "Show/Hide Sources" 切换来源面板
- [ ] 输入框支持多行输入（Shift+Enter 换行）
- [ ] 输入框支持 Enter 提交
- [ ] 流式响应时输入框禁用
- [ ] 响应完成后输入框重新启用

### 10. 错误处理测试
**测试场景：后端未运行**
- [ ] 停止后端服务
- [ ] 尝试创建会话或提交查询
- [ ] 显示友好的错误消息
- [ ] 不会导致页面崩溃

### 11. 响应式设计测试（可选）
- [ ] 调整浏览器窗口大小
- [ ] 布局自适应调整
- [ ] 小屏幕下侧边栏可折叠
- [ ] 移动端视图正常工作

### 12. 性能测试
- [ ] 长对话（10+ 消息）滚动流畅
- [ ] 流式响应无卡顿
- [ ] 无内存泄漏（检查开发者工具）
- [ ] 页面加载时间 < 2 秒
